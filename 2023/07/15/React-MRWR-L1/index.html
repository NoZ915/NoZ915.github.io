1-8. Create the React Project

這裡直接參考另一堂課做得筆記

Step 1.
下載Node.js
Step 2.
接著就可以利用Node.js的套件管理工具npm進行安裝React的環境。首先在電腦桌面打開終端機（terminal）（亦即終端機目前所在檔案位置為桌面），並且於終端機中輸入： 
npx create-react-app my-app

此時npm便會幫你安裝且建置好一個React專案”my-app”。
PS “my-app”為專案名稱，可自行改成任意名稱。

Step 3.
利用cd進入到"my-app"的檔案位置。 
cd my-app
Step 4.
最後在終端機打上`npm start`，便可以透過http://localhost:3000（預設為3000）打開"my-app"專案： 
npm start

當有跳出React的網頁（如以下畫面）即成功建置React專案！

PS 若想離開、關掉，則在終端機按下ctrl+c
